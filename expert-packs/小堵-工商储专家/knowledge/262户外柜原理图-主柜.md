### 原理图/流程图

- **电源输入**：AC380V 3P+N+PE 电网，通过 TAa^c 300/5 变压器接入。
- **主回路**：
  - L1, L2, L3, N 经过 QF 断路器后分为多个支路。
  - 支路一：L1, L2, L3, N → QF1 → 液冷机组
  - 支路二：L1, L2, L3, N → FU 熔断器 → L11, L12, L13, N1 → QF2 → FS（隔离开关）→ PE 接地 → 消防
  - 支路三：L1, L2, L3, N → L21, N2 → 电度表
  - 支路四：L1, N → 1QF1 → SP（整流模块）→ DC+, - → 2QF1 → 除湿机、水浸、检修插座
  - 支路五：L1, N → 1QF2 → 检修插座
- **直流系统**：
  - 由 SP 整流模块输出 DC+ 和 DC-，连接至 PCS（储能变流器）。
  - PCS 连接至 314Ah 电池模组（IP52S），包含 BAT1 至 BAT260。
  - 电池组通过虚线连接至消防系统。
- **其他负载**：
  - SQ 开关控制柜内照明。
  - 电度表用于计量用电量。
  - 水浸检测装置接入直流系统。
  - 检修插座提供维护电源。

---

### 接线图描述

- **交流侧**：
  - 电网输入为 AC380V 3P+N+PE。
  - 主母线 L1, L2, L3, N 从左侧引入，经 QF 后分出多条支路。
  - L1, L2, L3, N 分别接入不同负载。
- **直流侧**：
  - 通过 SP 整流模块将交流转换为直流。
  - 输出端标记为 DC+ 和 DC-，连接至 2QF1 和 PCS。
  - PCS 与电池组 BAT1-BAT260 相连，构成储能系统。
- **接地**：
  - PE 接地线从电网引出，连接至系统各部分的接地端。
- **端子标识**：
  - L1, L2, L3, N：三相加中性线。
  - L11, L12, L13, N1：分支线路编号。
  - L21, N2：另一分支线路。
  - DC+, -：直流正负极。
  - BAT1-BAT260：电池组编号。

### 3. 原理图/流程图：流向与功能描述

#### 通讯网络结构
- **EMS → LAN1 → JHJ**：站级EMS至站级交换机
- **LC → LAN2 → JHJ**：本地控制器至站级交换机
- **LC → LAN3 → PCS**：本地控制器至PCS
- **EMS → LAN4 → XT3**：站级EMS至面板网口
- **JHJ → XT3 (从柜1-3)**：站级交换机至各从柜，通过RJ45连接

#### 控制与监测回路
- **防逆流表**：通过1A1/1B1连接EMS，用于防止电流逆流
- **防过载表**：通过1A2/1B2连接EMS，检测过载状态
- **防超需表**：通过1A3/1B3连接EMS，监控用电需求
- **备用通道**：1A4/1B4作为备用信号通道
- **液冷机组**：由LC控制，经A1/B1接入LCU
- **水浸检测**：WS传感器，黄/蓝线接入A2/B2
- **除湿机**：CSJ，A/B端子接入A3/B3
- **电表**：PJ，21/22端子接入A4/B4

#### CAN总线通信
- **XFT1-XFT5**：五个CAN通信模块，分别连接PACK1-PACK5消防系统
- 每个XFT模块包含：
  - CAN-H（高电平）
  - CAN-L（低电平）
- 所有XFT模块均通过H2/L2与LC连接，形成CAN总线网络
