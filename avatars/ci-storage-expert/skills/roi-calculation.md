# 收益测算

> **级别**：🟢 基础
> **版本**：v1.0

---

## 技能说明

计算工商储项目的投资收益，输出 IRR/NPV/回收期等核心财务指标。

## 触发条件

当用户提到"收益测算"、"算一下收益"、"回收期"、"IRR"、"投资回报"时触发。

## 输入

| 参数 | 是否必须 | 说明 | 示例 |
|------|---------|------|------|
| 储能容量 | 必须 | 功率(kW)/容量(kWh) | 500kW/1MWh |
| 所在地区 | 必须 | 查电价用 | 广东广州 |
| 投资金额 | 建议提供 | 总投资(万元)，不提供则按行业均价估算 | 150万 |
| 充放电策略 | 可选 | 一充一放/两充两放 | 两充两放 |
| 用电信息 | 可选 | 月用电量/最大需量 | 5万kWh/月 |

## 执行流程

### Step 1：确定计算参数

**投资参数**：
- 系统单价：如未提供，按行业均价 [X] 元/Wh 估算（标注为估算值）
- 安装调试费：投资额的 5-8%
- 总投资 = 设备费 + 安装调试费

**运行参数**（默认值，可被用户覆盖）：
- 系统综合效率：88%
- 年可用天数：350 天
- 年衰减率：2.5%
- DOD（放电深度）：90%
- 项目周期：10 年

**成本参数**：
- 年运维成本：总投资的 2%
- 保险费：总投资的 0.3%

### Step 2：计算年度收益

**峰谷套利收益**：
```
日放电量 = 容量(kWh) × DOD × 系统效率
日收益 = 日放电量 × (峰电价 - 谷电价) × 充放次数
年收益 = 日收益 × 年可用天数
```

**需量电费节省**（如有用电数据）：
```
月需量削减 = min(储能功率, 原最大需量 × 削减比例)
月节省 = 需量削减(kW) × 需量电价(元/kW)
年节省 = 月节省 × 12
```

**需求响应补贴**：
```
年补贴 = 响应功率(kW) × 补贴单价(元/kW) × 年响应次数
```

### Step 3：计算财务指标

**年净收益** = 峰谷套利 + 需量节省 + 需求响应 - 运维成本 - 保险费

**静态回收期** = 总投资 ÷ 首年净收益

**IRR**：计算 10 年期内部收益率（考虑年衰减）

**NPV**：贴现率 7%，计算 10 年净现值

### Step 4：敏感性分析

对以下变量做 ±20% 敏感性分析：
- 峰谷电价差
- 系统效率
- 投资成本
- 年衰减率
