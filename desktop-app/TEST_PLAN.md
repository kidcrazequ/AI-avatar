## 第一阶段：构建测试

### 1.1 开发环境构建
**目标**：验证开发环境能否正常启动

**测试步骤**：
```bash
# 清理旧构建
rm -rf dist/ dist-electron/

# 编译 Electron 主进程
npm run build

# 检查输出
ls -la dist/
ls -la dist-electron/
```

**预期结果**：
- ✓ `dist/` 目录包含：index.html + assets/
- ✓ `dist-electron/` 目录包含：main.js, preload.js, soul-loader.js（及对应 .map 文件）
- ✓ 无编译错误
- ✓ 无 TypeScript 类型错误

**验证点**：
- [ ] dist/ 目录结构正确
- [ ] dist-electron/ 目录结构正确
- [ ] 所有 .js 文件都有对应的 .map 文件
- [ ] 构建日志无错误
