# 远景能源 ENS-L262 工商业储能一体机

> 数据来源：ENS-L262-01用户手册 V1（PDF，42页）

## 产品概述

- **型号**：ENS-L262（150kW/313.5kWh）
- **产品形态**：工商业储能一体柜（户外柜）
- **集成内容**：储能变流系统(PCS) + 电池 + BMS + 液冷温控 + 配电 + 消防
- **防护等级**：IP54（适应户外）
- **应用场景**：削峰填谷、能源消纳、应急备电、动态增容
- **支持并联**：多台柜体可直接并联扩容

## 系统架构

系统由两大部分组成：
- **储能电池系统**：313.5kWh 电池簇 + 灭火系统 + 温控单元
- **储能变流系统**：150kW PCS

系统示意（图3-1）：储能电池系统(A，含PCS) ↔ 电网(B) ↔ 负载(C)

### 系统接线架构（图6-3）
- 电网/负载 → PCS → 高压箱(含BCU) → 电池簇
- 配电模块 → EMS（485总线）→ BCU
- SPD（防雷）接入配电模块
- 液冷机组 & 消防系统 通过管路连接
- 各组件间通过 485 总线通信

## 核心参数

### PCS 储能变流器
- **额定功率**：150kW
- **最大效率**：>98.8%
- **拓扑**：高效三电平
- **特性**：
  - 模块化设计，可方便扩容
  - 内置直流预充电功能，即插即用
  - 中点电位平衡控制，减小纹波电压
  - 支持多台交流并联
  - 支持故障录波/远程在线升级
  - 风机模块独立设计

### PCS 指示灯状态
- **运行输出功率**：长亮绿灯
- **待机（0kW运行）**：绿灯 0.5s 快闪
- **未开机且无故障**：绿灯 1s 慢闪
- **故障**：长亮红灯

### 拨码开关说明
- **ADDR**：模块地址设置（二进制），第4位为最低位（从右往左进位）
- **120Ω**：4位二进制拨码，从左到右为第1~4位
  - 第1位：ARM_CAN1，预留CAN通信匹配电阻
  - 第2位：ARM_CAN2，预留CAN通信匹配电阻
  - 第3位：SYNC_CAN1，PARA中CAN信号匹配电阻，PCS并机通信用
  - 第4位：DSP_CAN2，凤凰端子中外部BMS的CAN信号匹配电阻

### 电芯参数
- **类型**：磷酸铁锂(LFP)
- **单体容量**：315Ah
- **额定电压**：3.2V
- **电压范围**：2.5~3.65V
- **尺寸**：(174.7±0.5)mm × (71.7±0.5)mm × (204.47±0.5)mm
- **重量**：5.6±0.2kg
- **外形**：方形铝壳

### 电池模块
- **型号**：ENS152LIC03
- **额定能量**：52.416kWh
- **充放电倍率**：0.25/0.5P
- **组合方式**：1P52S（52个电芯串联）
- **电芯类型**：方形铝壳
- **关键部件**：52个电芯 + 1个BMU
- **尺寸**：~D1140×W790×H245mm
- **重量**：325kg
- **BMS特性**：电池管理专用芯片，菊花链通信方式，支持电压/温度采集及均衡控制

### 电池簇
- **总容量**：313.5kWh（6个电池模块串联）
- **结构**：电芯 → 电池模块(1P52S) → 电池簇(串联)

## 温控系统

- **方式**：液冷（50%乙二醇冷却液）
- **液冷机组位置**：柜体顶部
- **原理**：风冷型制冷循环 → 板式换热器 → 乙二醇循环 → 电池液冷板
- **组件**（图3-10冷却机组原理图）：
  1. 压缩机
  2. 冷凝器
  3. 外循环风机
  4. 节流元件（电子膨胀阀）
  5. 板式换热器
  6. 循环水泵
  7. 电加热器
- **散热气流**：PCS采用前进风、后出风设计（图3-3-2）

## 消防系统

- **灭火剂**：全氟己酮
- **多级防护**：
  - **Pack级**：电池包内安装全氟己酮灭火装置，热敏线启动（≥170°C），无需电源
  - **簇级**：柜门侧上方安装簇级全氟己酮装置（1台4kg全氟己酮）
    - 电启动：烟感+温感同时检测异常且门禁闭合时触发
    - 热敏线启动：柜内温度≥170°C时自动触发
  - **水消防（选配）**：可选配水消防接口（柜体后面H接口）
- **探测器**：一氧化碳、氢气、VOC、感烟感温复合探测器（5合1）
  - 柜内安装1个复合探测器
  - 每个电池包内各安装1个探测器
- **气体循环**：CO≥190ppm 或 H₂≥400ppm 时启动换气；CO≤64ppm 且 H₂≤64ppm 时关闭
- **泄爆装置**：柜体后门上设有泄爆装置，热失控时触发，避免爆炸
- **窒息危险警告**：设备自动灭火时会产生大量气体，灭火过程中请勿靠近户外柜

## 安装要求

### 场地要求
- 环境干燥、通风良好、远离易燃易爆区域
- 土壤相对密实度≥98%
- 地基高出地面约300mm（防雨水）
- 安装现场应建造排水系统
- 勿在安装场地近距离种植树木（防风刮树枝堵塞进风口）

### 安装空间
- **单台**：前门预留开门及电池拆装≥1500mm，后面预留通风≥300mm
- **多台并排**：机柜间距≥10mm，前后预留同上；两排对开时两排间距≥1500mm+1500mm

### 固定方式
- **方法1**：螺栓固定 — M16×80mm螺栓固定在底部电缆沟形式地基上（底座预留安装孔φ18mm）
- **方法2**：焊接固定 — 焊接至槽钢/预制仓金属地板，注意防锈处理

### 运输
- **叉车**：承载力≥5t（额定起重量≥5000kg），插脚长度≥1800mm，通过底部叉孔运输
- **起吊**：使用柜体四个吊环，吊绳与水平夹角45-60°，操作区域5~10m内严禁站人

### 存储
- 温度：-30°C~+50°C（推荐-10°C~+25°C）
- 湿度：0~95%无冷凝
- 超过6个月需充放电使SOC达30%~40%
- 至少每半月巡检一次

### 地基要求
- 混凝土基础或钢结构基础（图5-2/5-3）
- 基坑底部夯实填平
- 地基高出水平地面约300mm
- 需考虑线缆走线、排水、维护平台
- 预埋电缆导管，两端暂时密封
- 连接电缆后用耐火泥密封防啮齿动物

### 接口规格（表6-1）
| 接口 | 描述 | 推荐规格 |
|------|------|---------|
| L1/L2/L3 | PCS交流侧至电网侧 | 4×70+1×35mm²/0.6/1kV |

| PE | 接地线 | ≥35mm²铜线或相同载流量接地扁钢 |

### 接地
- 接地电阻≤4Ω
- **推荐**：热镀锌扁钢焊接（搭接面积40×70mm），完成后喷涂整个固定面
- **备选**：35mm²铜接地线缆，M10螺栓固定

### 五大安全法则（电气操作必须遵守）
1. 断开储能系统所有外部连接及内部供电电源
2. 确保各断开处不会被意外重新上电
3. 使用万用表确保设备内部已完全不带电
4. 施行必要的接地连接
5. 对操作部分临近可能带电部件进行绝缘遮盖

### 上电顺序
闭合：1QF1 → 1QF2 → QF → QF1 → QF2

### 下电顺序
通过EMS等上位机控制系统下发停机指令，确认停机后：
断开：QF1 → QF2 → QF → 1QF1 → 1QF2

### 电网制式
- 支持 TN-S 制式（L1/L2/L3/N/PE 五线制）
