### 3. 原理图/流程图：流向与部件功能

#### 流向描述：

- **直流输入路径**：
  - 电池PACK（261kWh，5个IP52S 314Ah的PACK）通过B+和B-引出。
  - 经过QF（直流断路器）进行过流保护。
  - 再经过FU（熔断器）进一步保护。
  - 进入DC Contactor（直流接触器），由控制系统控制通断。
  - 最终接入PCS的直流侧输入端（INV模块）。

- **交流输出路径**：
  - INV模块将直流电转换为交流电。
  - 输出经由AC Relays（交流继电器）控制通断。
  - 每相（A、B、C）均串联电感L进行滤波。
  - 各相再经过FU（熔断器）进行短路保护。
  - 输出至L11、L12、L13（三相火线）和N1（中性线）。

- **控制与安全**：
  - EPO-1和EPO-2为紧急停机按钮，可手动切断系统电源。
  - J1~J10为接线端子，可能用于信号或辅助电源连接。

#### 主要部件功能：

| 部件 | 功能 |
|------|------|
| PACK | 提供储能，容量261kWh，由5个314Ah电池组成 |
| QF | 直流断路器，保护电路免受过载或短路影响 |
| FU | 熔断器，提供过电流保护 |
| DC Contactor | 控制直流侧通断，配合电池管理系统使用 |
| INV | 双向逆变器，实现直流与交流之间的能量转换 |
| AC Relays | 控制交流输出的通断 |
| L | 电感，用于平滑交流输出电流，减少谐波 |
| EPO-1/EPO-2 | 紧急停机按钮，确保系统可在紧急情况下快速停止运行 |
| J1~J10 | 接线端子，用于连接外部控制信号或通信线路 |
