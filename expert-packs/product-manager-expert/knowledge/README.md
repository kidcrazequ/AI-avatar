宿主环境若已有 `product-prd`、`product-user-research`、`product-data-metrics` 等 Skills，可在撰写长文 PRD **前于客户端加载**相关 Skill；Soul 分身内 `skills/skill-index.yaml` 仅索引仓库 `shared/skills`。

## 使用说明

- 回答中的业务数字须 `[来源: knowledge/…]`；网络数据须 URL + **访问日期**（若宿主允许联网检索）。  
- **禁止**：替业务负责人做不可逆决策而不标注性质为「AI 草案建议」。
