# 工商储产品质量指标查询 · query-product-quality

> **级别**：[■■] 进阶
> **版本**：v1.0
> **最后更新**：2026-04-20

## 技能说明

本技能专用于查询 `00_工商储-产品质量指标dashboard_260303` Excel 文件的质量数据，给出固定的 `query_excel` 参数模板，**避免同一问题因参数不一致导致查到不同数据（图漂）**。

查询完成后，必须调用 `load_skill('draw-chart')` 并按其规范输出图表，不允许只用文字描述。

---

## 数据源约定

| 参数 | 固定值 |
|---|---|
| `file` | `00_工商储-产品质量指标dashboard_260303` |
| `sheet`（原始数据）| `总原始表` |
| 统计周期格式 | YYMM 整数（如 `2601` = 2026年1月，`2603` = 2026年3月） |

> ⚠️ `"月度"` sheet **不存在**，禁止使用。

---

## 参数模板（**格式化问题必须逐字照抄**）

当用户问题匹配下列格式时，**禁止自由发挥**，必须按给定模板填槽调用 `query_excel`，否则同问多次会得到不同参数 → 查到不同数据 → 图漂：

**格式 A：`N 机型 YYYY 年 M1~M2 月 X 指标`**（例："215 机型 2026 年 1~3 月设备侧效率折线图"）

```
query_excel({
  "file": "00_工商储-产品质量指标dashboard_260303",
  "sheet": "总原始表",
  "filter": {
    "机型": "<N>",
    "统计周期": {"$in": [<YYMM1>, <YYMM2>, <YYMM3>]}
  },
  "columns": ["统计周期", "项目", "<X 指标中文原名>"]
})
```

**格式 B：`N 机型 YYYY 年 X 指标`**（没给月区间，按全年）

```
query_excel({
  "file": "00_工商储-产品质量指标dashboard_260303",
  "sheet": "总原始表",
  "filter": {
    "机型": "<N>",
    "统计周期": {"$gte": <YY01>, "$lte": <YY12>}
  },
  "columns": ["统计周期", "项目", "<X 指标中文原名>"]
})
```

**以上两条模板的存在目的**：让"N 机型 YYYY 年 M~M 月 X 指标"这类问题每次跑出字符一致的 tool 参数，配合 chatStore 的 `normalizeQueryExcelArgs` cache 达到"同问同答"。偏离模板的写法本身合法但会破坏缓存命中，**仅当用户问题不符合格式 A/B 时**才允许自由构造参数。

---

## 填槽硬规矩（每一条违反都会破坏 cache 命中）

- `<N>` 取用户原话里的机型数字（`215`/`372` 等），**不加单位、不加"机型"两字、不加引号外再套引号**
- `统计周期` 用 YYMM 整数格式（如 `2601` = 2026年1月；`2603` = 2026年3月），**不要用 "2026-01" 字符串**
- 多个月份用 `{"$in": [2601, 2602, 2603]}` 形式；单月也用 `{"$in": [2601]}`，**不要降级成标量**
- `columns` 固定三列 `["统计周期", "项目", "<指标>"]`，不要额外追加其他列
- `"sheet"` 固定写 `"总原始表"`；**不要写 `"月度"`（该 sheet 不存在）**

---

## 多项目场景说明

`总原始表` 中同一 `统计周期` 通常有**多个项目行**（如 215 机型 2601 月有 10+ 个项目）。这是正常的，不是查询失败。画图时应把**所有非 null 行**都纳入：

- 按项目展示（各项目一个柱/线）
- 或按 `统计周期` 分组展示平均值趋势

**不允许只返回单一项目的值。**
