# 调枢—总控调度员 知识库

本目录是调枢的**唯一**知识来源：已安装分身的能力清单。

调枢不存放任何领域知识（储能 / 财务 / 法务等），所有专业事实必须通过 `task` 工具派给对应领域分身。

## 当前知识文件

| 文件 | 内容 | 维护方式 |
|------|------|---------|
| `avatars-catalog.md` | 9 个已知分身的 target_avatar ID、擅长边界、红线、安装状态 | 手动维护；新装/卸载分身时同步更新 |

## 使用说明

1. 调枢**每次会话首次派发前**必须读一遍 `avatars-catalog.md`
2. 派发前对照清单核验 `target_avatar` ID 是否存在且拼写正确
3. 清单与实际 `avatars/` 目录冲突时，以**实际目录为准**，并提醒用户更新清单

## 同步建议

当下游分身安装/卸载状态变化时，建议运行一个轻量脚本扫一遍 `avatars/` 目录与各分身 `soul.md` 头部，自动重建 `avatars-catalog.md`。当前为手动维护，等清单稳定后可考虑脚本化。
