### 产品要求

#### 4.1 一般要求

- **4.1.1** 储能系统柜采用一体化集成设计，将储能电池簇（含 BMS）、储能变流器等设备安装在户外储能柜内，储能柜拥有独立的配电系统、温度控制系统、消防系统等自动控制和安全保障系统，储能系统采用 AC400V 接入。
- **4.1.2** 高低压及信号线分开布置，强、弱电二次回路分开敷设，每个功率接线端子只接一根导线，最大程度减少各类电磁干扰；
- **4.1.3** 要求储能设备采用模块化设计，储能电池电芯-电池模组-电池簇模块化层级，层次分明、结构清晰、功能完善，应包含完善的电池、电池管理系统（BMS）、温控系统（含空调或液冷机组）、自动灭火系统、绝缘监测、接地保护等。
- **4.1.4** 储能系统应具备完善的保护功能，包括但不限于电池本体保护、电池过流过压保护、短路保护、并网保护等。
- **4.1.5** 储能系统的运行要求：储能系统自身运行控制系统应提供完善的内部监测、控制、故障保护与切除、事件记录功能，包括但不限于投切控制、运行模式控制、设备状态、运行温度和监测等功能。
- **4.1.6** 电池模块和电池簇内，电气间隙和爬电距离、绝缘电阻、介损强度应满足 NB / T 42091-2016《电化学储能电站用锂离子电池技术规范》。
- **4.1.7** 为避免因单体电池特性差异较大而引起整组电池性能和寿命下降，乙方设备应具备保证电池指标的均衡性解决措施。
- **4.1.8** 根据工程需要可以召开设计联络会或以其它形式解决设计制造中的问题。文件交接要有记录，设计联络会议应有纪要。未尽事宜，双方协商处理，可以以其它形式补充。以后协调所形成的文件与规范书具有同等效力。

### 3. 原理图/流程图

- **能量流向**：
  - IPS2S模块（电池包） → SBCU → EMU → PCS → AC400V电网
  - 反向充电路径：AC400V电网 → PCS → EMU → SBCU → IPS2S模块
- **控制信号流**：
  - SBCU ↔ EMU（通信：RS485）
  - EMU ↔ Power Control Unit（通信：RS485）
  - EMU ↔ PCS（通信：CAN）
  - EMU ↔ Heating and Cooling System（控制信号）
- **安全与监测**：
  - Real Detector、Smoke Detector、Aerosol-Generating Extinguisher 连接至Power Control Unit，用于火灾预警与灭火
  - LED 由Power Control Unit驱动，显示运行状态
- **关键功能部件说明**：
  - **SBCU**：电池管理系统主控单元，管理多个IPS2S电池组
  - **EMU**：能量管理单元，协调充放电过程，与PCS通信
  - **PCS**：功率转换系统，实现直流到交流的双向转换
  - **Pre-charge**：预充电回路，防止冲击电流
  - **Heating and Cooling System**：温控系统，维持电池工作温度

  5. 上行量：储能装置上传告警信息、模拟信息至后台监控系统；
  6. 下行量：后台监控系统下达遥控信息至储能装置；
  7. PCS 应具备故障录波功能，不少于 200 条存储记录。

### 4.3.2 其他功能需求

- **有功功率控制功能**：PCS 储能装置可根据微电网运行控制系统指令控制其有功功率输出。电池储能系统接收并实时跟踪执行微电网运行控制系统发送的有功功率控制信号，根据控制指令自动调节有功输出，输出有功功率与设置值偏差不超过 3%。
- **电压/无功调节功能**：PCS 储能装置可根据微电网运行控制系统控制指令等信号实时跟踪调节无功输出，参数如无功功率、功率因数等可由微电网运行控制系统远程设定。
- **主动孤岛检测功能**：孤岛状态下，能够在 2s 内检测出孤岛状态，同时通过监控系统上报故障信息。
- **耐受电压异常能力**：交流输出端三相电压的允许偏差为额定电压的 +15%、-15%（可设置）。

  - 消防系统通过DO（数字输出）由主控单元控制。
  - 主控单元通过RS485与上位机通信。
  - 上位机通过ETH或4G连接至Web平台，实现远程监控。

- **数据流向**：
  - BMS → CAN → 主控单元：实时传输电池电压、电流、温度等参数。
  - PCS → CAN/RS485 → 主控单元：传输充放电状态及功率信息。
  - 液冷水机 → CAN → 主控单元：反馈冷却系统运行状态。
  - 气体探测器、温感、烟感 → DI → 主控单元：上传环境监测信号。
  - 主控单元 → DO → 消防：在报警时触发消防动作。
  - 主控单元 → RS485 → 上位机 → Web：上传系统运行数据与报警信息。

- **功能逻辑**：
  - 实时监测电池热电参数（电压、电流、温度）。
  - 计算并显示SOC、SOH、DOD等状态参数。
  - 根据荷电状态进行充放电控制，防止过压、过流。
  - 故障诊断与保护：过充、过放、短路、反接、过载、过温保护。
  - 异常时自动隔离故障电池，并上报告警。
