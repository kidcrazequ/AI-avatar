### 原理图/流程图：描述流向、各部件名称和功能

该图为液冷机策略的控制逻辑流程图，用于根据电池温度条件自动调节液冷机的工作模式（制冷、制热、循环或停止）。流程图从“开始”节点启动，依次进行参数读取与判断。

#### 流程结构描述：

1. **起始节点**：
   - “开始” → 读取参数

2. **初始判断**：
   - 判断“系统运行时间 ≤ 0.3h”？
     - 若“否”：进入主逻辑判断
     - 若“是”：直接跳转至“液冷机停止”

3. **主逻辑分支**（基于电池温度）：
   - 判断“电芯Tmax > 45℃”？
     - 是：进入“液冷机启动制冷模式”
       - 进一步判断“电芯Tmax ≥ 30℃”？
         - 否：返回“液冷机停止”
         - 是：继续执行制冷
     - 否：进入下一个判断
   - 判断“电芯Tmin + 5℃”？
     - 是：进入“液冷机启动制热模式”
       - 进一步判断“电芯Tmin  30℃”？
     - 是：进入“液冷机启动制冷模式”
       - 进一步判断“电芯Tmax ≥ 25℃”？
         - 否：返回“液冷机停止”
         - 是：继续执行制冷
     - 否：进入中间判断
   - 中间判断：“电芯Tmax - 电芯Tmin > 8℃”？
     - 是：进入“液冷机启动循环模式”
       - 进一步判断“电芯Tmax - 电芯Tmin > 5℃”？
         - 否：返回“液冷机停止”
         - 是：继续执行循环
     - 否：进入“液冷机停止”

4. **最终状态**：
   - 所有路径最终汇聚到“液冷机停止”节点，表示当前阶段操作结束。

#### 可设参数说明：

| 类别 | 参数项 |
|------|--------|
| 一、闲时 | 制冷开启温度、制冷停止温度、制热开启温度、制热停止温度 |
| 二、工作时 | 制冷开启温度、制冷停止温度、制热开启温度、制热停止温度 |
| 三、全工况 | 循环开启温差、循环停止温差、提前运行时间 |

**备注**：除运行时间外，所有判定延时10秒执行。

### 数据表格

| 类型 | 内容 |
|------|------|
| 控制模式 | 制冷模式、制热模式、循环模式、停止 |
| 温度阈值 | 45℃, 30℃, 25℃, 15℃, 10℃, 5℃ |
| 时间条件 | 系统运行时间 ≤ 0.3h |
| 延迟时间 | 所有判定延时10秒执行（除运行时间外） |
| 判断条件 | 电芯Tmax > 45℃, 电芯Tmin + 5℃, 电芯Tmin  30℃, 电芯Tmax - 电芯Tmin > 8℃, 电芯Tmax - 电芯Tmin > 5℃ |
