## 系统架构

```
工商业储能系统架构

┌──────────────────────────────────────────────┐
│                   EMS 能量管理系统              │
│         策略管理 / 收益优化 / 监控告警          │
├──────────────────────────────────────────────┤
│                   PCS 储能变流器               │
│              DC-AC 变换 / 功率控制              │
├──────────────────────────────────────────────┤
│                   BMS 电池管理系统              │
│         SOC 估算 / 均衡管理 / 安全保护          │
├──────────────────────────────────────────────┤
│                   电池组 (Pack/Rack)           │
│              电芯 → 模组 → 电池簇              │
├──────────────────────────────────────────────┤
│               温控系统 (风冷/液冷)              │
├──────────────────────────────────────────────┤
│            消防系统 / 电气保护 / 柜体           │
└──────────────────────────────────────────────┘
```
