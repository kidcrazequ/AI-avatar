## 建议后续收录（占位说明）

以下内容请由业务方自行从**财政部会计准则、国家税务总局公告原文、公司披露的定期报告**等转化而来，并按版本管理：

| 建议子目录 | 用途 |
|-----------|------|
| `会计准则/` | 与您主体适用准则相关的节选 + 生效日期 |
| `税法实务/` | 与您行业相关的征管口径笔记（标注文号页码） |
| `报表模板/` | 管理口径三大表、预算滚动表样板 |
| `合规清单/` | 内控测试表、纳税申报勾稽自检项 |

官方公开资料可从以下来源对齐目录（示例，非穷尽）：

- 财政部会计司发布的《企业会计准则》及应用指南（请下载您需要的具体准则 PDF 后人工摘录）  
- 国家税务总局发布的政策解读与公告

## 使用说明

1. 新增 `.md` 后**必须**更新本表。  
2. 分身回答须写 `[来源: knowledge/文件名]`。  
3. **禁止**：让 LLM「生成一整份准则」填入本目录冒充权威文本。
