# ES125-2W温度控制策略大纲

## 原理图/流程图：描述流向、各部件名称和功能

该图为ES125-2W系统的温度控制策略流程图，展示了系统在不同运行模式下的温度监测与液冷机控制逻辑。流程从“开始”节点出发，依次执行以下步骤：

1. **采集温度数据**：系统首先采集当前储能系统的电芯温度数据。
2. **判断储能系统运行模式**：
   - 若为“是”（系统运行），进入系统运行分支；
   - 若为“否”（系统静置或待机），进入静置/待机分支。
3. **系统运行分支**：
   - 判断最小电芯温度 $T_{cell,min}$ 是否 ≤ 电芯预设值21：
     - 是：启动液冷机加热模式；
     - 否：继续判断最大电芯温度 $T_{cell,max}$ 是否 ≥ 电芯预设值23。
       - 是：启动液冷机制冷模式；
       - 否：无操作。
   - 加热模式关闭条件：当 $T_{cell,min} \geq$ 电芯预设值22 时，关闭加热模式。
   - 制冷模式关闭条件：当 $T_{cell,min} \leq$ 电芯预设值24 时，关闭制冷模式。
4. **系统静置或待机分支**：
   - 判断 $T_{cell,min} \leq$ 电芯预设值11：
     - 是：启动液冷机加热模式；
     - 否：判断 $T_{cell,max} \geq$ 电芯预设值13：
       - 是：启动液冷机制冷模式；
       - 否：无操作。
   - 加热模式关闭条件：当 $T_{cell,min} \geq$ 电芯预设值12 时，关闭加热模式。
   - 制冷模式关闭条件：当 $T_{cell,min} \leq$ 电芯预设值14 时，关闭制冷模式。
5. 所有路径最终返回至“采集温度数据”，形成闭环控制。

# ES125-2L温度控制策略大纲

## 原理图/流程图：描述流向、各部件名称和功能

该图为ES125-2L系统的温度控制策略流程图，用于根据储能系统运行状态及电芯温度数据，决定是否启动空调加热、制冷或除湿模式，以及BLMU风扇的启闭。

### 流程概述

1. **开始** → 采集温度数据
2. 判断储能系统运行模式：
   - 若为“是”（系统运行）→ 进入系统运行分支
   - 若为“否”（系统静置或待机）→ 进入静置/待机分支

---

### 分支一：系统静置或待机模式

- 判断条件：`T_cell,max ≥ 电芯预设值1` 或 `ΔT_cell ≥ 温差预设值1`
  - 是 → BLMU风扇开启
  - 否 → BLMU风扇关闭

- 判断条件：`T_cell,min ≤ 电芯预设值11`
  - 是 → 启动空调加热模式
    - 当 `T_cell,min ≥ 电芯预设值12` 时，关闭空调加热模式
  - 否 → 继续判断下一个条件

- 判断条件：`T_cell,max ≥ 电芯预设值13`
  - 是 → 启动空调制冷模式
    - 当 `T_cell,min ≤ 电芯预设值14` 时，关闭空调制冷模式
  - 否 → 继续判断下一个条件

- 判断条件：满足除湿条件
  - 是 → 启动空调除湿
    - 判断是否满足湿度预设值
      - 满足 → 结束当前流程
      - 不满足 → 继续循环

---

### 分支二：系统运行模式

- 判断条件：`T_cell,max ≥ 电芯预设值2` 或 `ΔT_cell ≥ 温差预设值2`
  - 是 → BLMU风扇开启
  - 否 → BLMU风扇关闭

- 判断条件：`T_cell,min ≤ 电芯预设值21`
  - 是 → 启动空调加热模式
    - 当 `T_cell,min ≥ 电芯预设值22` 时，关闭空调加热模式
  - 否 → 继续判断下一个条件

- 判断条件：`T_cell,max ≥ 电芯预设值23`
  - 是 → 启动空调制冷模式
    - 当 `T_cell,min ≤ 电芯预设值24` 时，关闭空调制冷模式
  - 否 → 继续循环

---

### 主要组件及其功能

| 组件名称 | 功能描述 |
|--------|---------|
| 采集温度数据 | 获取电芯最高温 `T_cell,max`、最低温 `T_cell,min` 和温差 `ΔT_cell` |
| 储能系统运行模式 | 判断系统处于运行还是静置/待机状态 |
| BLMU风扇 | 根据温度条件控制风扇启闭，用于散热 |
| 空调加热模式 | 当电芯温度过低时启动加热 |
| 空调制冷模式 | 当电芯温度过高时启动制冷 |
| 空调除湿 | 在满足特定湿度条件下启动除湿功能 |

---

### 控制逻辑参数汇总表

| 参数名称 | 用途说明 | 出现位置 |
|--------|--------|--------|
| 电芯预设值1 | 静置/待机模式下高温阈值 | `T_cell,max ≥ 电芯预设值1` |
| 温差预设值1 | 静置/待机模式下温差阈值 | `ΔT_cell ≥ 温差预设值1` |
| 电芯预设值11 | 静置/待机模式下低温启动加热阈值 | `T_cell,min ≤ 电芯预设值11` |
| 电芯预设值12 | 加热模式关闭阈值 | `T_cell,min ≥ 电芯预设值12` |
| 电芯预设值13 | 静置/待机模式下高温启动制冷阈值 | `T_cell,max ≥ 电芯预设值13` |
| 电芯预设值14 | 制冷模式关闭阈值 | `T_cell,min ≤ 电芯预设值14` |
| 电芯预设值2 | 运行模式下高温阈值 | `T_cell,max ≥ 电芯预设值2` |
| 温差预设值2 | 运行模式下温差阈值 | `ΔT_cell ≥ 温差预设值2` |
| 电芯预设值21 | 运行模式下低温启动加热阈值 | `T_cell,min ≤ 电芯预设值21` |
| 电芯预设值22 | 运行模式下加热关闭阈值 | `T_cell,min ≥ 电芯预设值22` |
| 电芯预设值23 | 运行模式下高温启动制冷阈值 | `T_cell,max ≥ 电芯预设值23` |
| 电芯预设值24 | 运行模式下制冷关闭阈值 | `T_cell,min ≤ 电芯预设值24` |

> 注：所有“电芯预设值”和“温差预设值”均为系统内部设定值，单位未标注，但通常为摄氏度（℃）或开尔文（K），具体数值需查阅设备手册。
