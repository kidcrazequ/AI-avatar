# 文档

“文档”在本知识库中指代结构化、可被AI分身直接读取和引用的领域知识载体，主要为 Markdown 格式的技术类文件（如用户手册、白皮书、作业指导书等），用于支撑工商储系统（如 ENS-L262-01）的安装、操作、维护与故障处理。其核心定位是**面向现场工程师的实用指南**，而非覆盖全部工程参数的技术规格书；因此内容聚焦安全操作与基础流程，主动省略大量需专业交付的细节（如电气耐压等级、CAN波特率、螺栓扭矩值等）。文档具有明确来源标识、命名规范及质量校验要求，并支持图文协同——PDF 原始文件经转换后，关键页面图片被提取至 `images/` 目录，其中数据须识别并写入对应 Markdown 文件。

| 属性 | 值 |
|------|-----|
| 主要格式 | Markdown（`.md`） |
| 原始来源 | PDF、DOCX 等格式的原始文件（如 `ENS-L262-01用户手册 -V1.pdf`） |
| 存储路径 | `knowledge/` 目录及其子目录 |
| 图片关联 | 关键页面以 PNG 形式存于 `knowledge/images/`，内容需识别后写入对应 Markdown |
| 命名规范 | 中文名称 + 简短后缀（如 `产品名-用户手册.md`），用 `-` 分隔，无空格 |
| 质量要求 | 开头注明原始文档名称与版本；关键数值标注单位；图片数据已写入文本；README 表格同步更新 |
| 功能定位 | 支持 AI 分身回答时溯源引用（格式：`[来源: knowledge/文件名]`），填补工程交付中的操作性知识缺口 |

相关实体：用户手册, 知识库, Markdown, PDF, images, 运维白皮书, 安装作业指导书
