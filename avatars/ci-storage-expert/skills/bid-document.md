# 标书辅助

> **级别**：🟡 进阶
> **版本**：v1.0

---

## 技能说明

根据招标要求，辅助撰写工商储项目技术方案/投标文件中的技术章节。

## 触发条件

当用户提到"标书"、"投标"、"技术方案撰写"、"招标响应"时触发。

## 输入

| 参数 | 是否必须 | 说明 | 示例 |
|------|---------|------|------|
| 招标要求 | 必须 | 招标文件/技术要求摘要 | 文件或关键要求描述 |
| 我方方案 | 建议提供 | 已有的方案设计 | 方案文档或关键参数 |
| 撰写章节 | 可选 | 指定章节 | "技术方案"章节 |

## 执行流程

### Step 1：分析招标要求
- 提取技术要求清单
- 识别评分重点和必须响应项
- 标注我方方案的响应程度（完全满足/部分满足/需说明）

### Step 2：撰写技术内容
- 按招标要求的结构组织内容
- 技术描述准确、参数与设备实际参数一致
- 突出差异化优势
- 对偏离项给出合理说明

### Step 3：质量检查
- 逐条核对招标要求是否全部响应
- 检查参数一致性
- 检查格式规范

## 输出格式

按招标文件要求的章节结构输出。通用结构：

```markdown
# 技术方案

## 1. 项目理解
[对项目需求的理解和分析]

## 2. 总体方案设计
### 2.1 设计原则
### 2.2 系统架构
### 2.3 技术路线

## 3. 系统详细设计
### 3.1 储能电池系统
### 3.2 PCS 变流系统
### 3.3 BMS 电池管理系统
### 3.4 EMS 能量管理系统
### 3.5 温控系统
### 3.6 消防系统
### 3.7 监控与通信

## 4. 设备清单及参数
[设备参数响应表]

## 5. 实施方案
### 5.1 项目组织
### 5.2 施工方案
### 5.3 进度计划

## 6. 质保与售后服务

## 7. 偏离说明表
```

## 质量标准

- [ ] 招标要求逐条响应
- [ ] 技术参数与实际设备一致
- [ ] 偏离项有合理说明
- [ ] 差异化优势有突出

## 安全红线

- **不编造设备参数**。所有参数必须与实际设备一致
- **不虚假承诺**。服务承诺必须可兑现
- 标书内容是**草稿**，**必须经用户审核后才能提交**
