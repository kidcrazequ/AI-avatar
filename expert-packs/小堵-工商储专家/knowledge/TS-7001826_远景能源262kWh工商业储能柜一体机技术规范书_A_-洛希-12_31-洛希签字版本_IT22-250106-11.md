### 产品要求

#### 4.1 一般要求

- **4.1.1** 储能系统柜采用一体化集成设计，将储能电池簇（含BMS）、储能变流器等设备安装在户外储能柜内，储能柜拥有独立的配电系统、温度控制系统、消防系统等自动控制和安全保障系统，储能系统采用AC400V接入。
- **4.1.2** 高低压及信号线分开布置，强、弱电二次回路分开敷设，每个功率接线端子只接一根导线，最大程度减少各类电磁干扰。
- **4.1.3** 要求储能设备采用模块化设计，储能电池电芯-电池模组-电池簇模块化层级，层次分明、结构清晰、功能完善，应包含完善的电池、电池管理系统（BMS）、温控系统（含空调或液冷机组）、自动灭火系统、绝缘监测、接地保护等。
- **4.1.4** 储能系统应具备完善的保护功能，包括但不限于电池本体保护、电池过流过压保护、短路保护、并网保护等。
- **4.1.5** 储能系统的运行要求：储能系统自身运行控制系统应提供完善的内部监测、控制、故障保护与切除、事件记录功能，包括但不限于投切控制、运行模式控制、设备状态、运行温度和监测等功能。
- **4.1.6** 电池模块和电池簇内，电气间隙和爬电距离、绝缘电阻、介质感度应满足NB/T 42091-2016《电化学储能电站用锂离子电池技术规范》。
- **4.1.7** 为避免因单体电池特性差异较大而引起整组电池性能和寿命下降，乙方设备应具备保证电池指标的均衡性解决措施。
- **4.1.8** 根据工程需要可以召开设计联席会或以其它形式解决设计制造中的问题。文件交接要有记录，设计联席会议应有纪要。未尽事宜，双方协商处理，可以以其它形式补充。以后协调所形成的文件与规范书具有同等效力。

### 3. 原理图/流程图

该图展示的是远景能源 ENS-L262 工商业储能一体机的电气系统原理图，分为强电和弱电两部分：

- **强电部分**：
  - 输入：交流电网（AC 380V, 3P4W）通过断路器 Q1 和接触器 KM1 接入。
  - 经过变压器 T1、整流桥（未标注名称）、直流母线（DC Bus）连接至逆变器模块（PCS）。
  - 电池组通过熔断器 F1 和接触器 KM2 连接至直流母线。
  - 逆变器输出接入并网开关（QF1）后接入电网。
  - 充放电控制由 PLC 控制，通过继电器和接触器实现通断。

- **弱电部分**：
  - 控制单元包括：PLC、EMS（能量管理系统）、BMS（电池管理系统）、HMI（人机界面）等。
  - BMS 负责监测电池状态（电压、电流、温度等），并与 EMS 通信。
  - EMS 与 PCS 通信，协调充放电策略。
  - HMI 提供操作界面，用于监控和设置参数。
  - 所有控制信号通过通讯总线（如 CAN 或 RS485）传输。

- **流向描述**：
  - 电网 → Q1 → KM1 → T1 → 整流 → DC Bus → 电池组（充电）或 PCS（逆变）
  - 电池组 → DC Bus → PCS → QF1 → 电网（放电）
  - 控制信号从 HMI → EMS → BMS / PCS → 各执行元件（接触器、继电器）

### 4.3.1 基本功能

1. PCS 变流器可以实现电网与电池间的交直流转换，完成两者间的双向能量流动，是储能系统的主要执行机构和核心组件。
2. 支持有功无功调节功能。
3. 监控要求：
   1) 遥测：装置输出电流，电网电压/电流，电网电压频率，功率因数等；
   2) 遥信：装置工作状态（运行/停止/故障），输入过欠压，过温，过流等；
   3) 遥控：开/关机。
   4) 遥调：功率及电流设定、运行模式设定等。
   - 上传量：储能装置上传告警信息、模拟信息等至后台监控系统。
   - 下行量：后台监控系统下达遥控信息至储能装置。
   5) PCS 应具备故障录波功能，不少于 200 条存储记录。

### 4.3.2 其他功能需求

PCS 储能装置应具备以下功能：

1. 有功功率控制功能，PCS 储能装置可根据微电网运行控制系统指令控制其有功功率输出。为实现有功功率调节功能，电池储能系统能接收并实时跟踪执行微电网运行控制系统发送的有功功率控制信号，根据微电网运行控制系统控制指令等信号自动调节有功输出，输出有功功率与设置值偏差不超过 3%。
2. 电压/无功调节功能，PCS 储能装置可根据、微电网运行控制系统控制指令等信号实时跟踪调节无功输出，其参数为无功功率、功率因数等参数可由微电网运行控制系统远程设定。
3. PCS 储能装置具备主动孤岛检测功能，孤岛状态下，防孤岛保护动作时间不大于 2s，同时通过监控系统上报故障信息。
4. PCS 储能装置具有一定的耐受电压异常能力。交流输出端三相电压的允许偏差为额定电压的 +15%、-15%（可设置）。
5. PCS 储能装置在并网模式下具备一定的耐受系统频率异常的能力，能够在电网频率偏离下运行（可设置）。

### 电压轮廓线图（低电压穿越）

#### 图一：B1 类和 B2 类储能变流器交流端口电压跌落曲线

- 横轴：时间 / s，范围从 -1 到 2.0
- 纵轴：交流端口电压 / 额定电压，范围从 0 到 1.0
- 关键点：
  - t = 0 时，电压为 1.0（额定电压）
  - t = 0.150 s 时，电压跌至 0.9（90% 额定电压）
  - t = 0.625 s 时，电压跌至 0.2（20% 额定电压）
  - t = 2.0 s 时，电压恢复至 0.9（90% 额定电压）
- 电压轮廓线为斜线连接点 (0, 1.0) → (0.150, 0.9) → (0.625, 0.2) → (2.0, 0.9)
- 储能变流器在电压轮廓线以上区域运行时，应不脱网运行。

#### 注释内容：

- 储能变流器交流端口发生三相短路故障和两相短路故障时，低电压穿越考核电压为交流端口线电压；
- 储能变流器交流端口发生单相接地短路故障时，低电压穿越考核电压为交流端口相电压。

#### B1 类和 B2 类储能变流器低电压穿越要求：

- 交流端口电压跌落至 0 时，储能变流器不脱网连续运行 150ms；
- 交流端口电压跌落至额定电压的 20% 时，储能变流器不脱网连续运行 625ms；
- 交流端口电压跌落至额定电压的 85% 时，储能变流器不脱网连续运行 2s；
- 交流端口电压跌落至电压轮廓线以下时，储能变流器与电网断开。

以下为原理图中各部件名称、功能及流向描述：

- 经预充电支路后，通过接触器接入电池组；
- 每个电池模组（共5个）包含BatS2和Bat1，连接至对应的采集均衡单元；
- 所有采集均衡单元通过均化链将数据上传至主控；
- 主控与EMS通过485通信进行交互；
- 分流器实时采集电流数据送入主控；
- 总压采集、接触器监测和续电流采集也接入主控。
