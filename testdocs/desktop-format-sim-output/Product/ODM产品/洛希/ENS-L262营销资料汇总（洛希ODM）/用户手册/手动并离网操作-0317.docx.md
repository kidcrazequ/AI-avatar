发布日期:2025 年 3 月 18 日

一、并离网支持数量

1.并网组串式支持 20 台。

2.并网集中式支持 8 台。

3.离网组串式支持 6 台。

4.离网集中式支持 3 台。

离网使用时，每个并网点需要 1 台主机，多个并网点不能协同工作，只能独立运 行。

二、配置检查

1.是否设置主机，需要设置为主机（1 为主机）

2.是否软起使能，需要勾选软起使能（勾选 Bit 15）

## 三、并网转离网操作流程 / Grid-Connected to Off-Grid Operation Procedure:

三、并网转离网操作流程 / Grid-Connected to Off-Grid Operation Procedure:

1.关机/Shutdown:

首先关闭所有设备，以确保在进行模式转换时不会出现不必要的电力波动。

First, shut down all devices to ensure no unnecessary power fluctuations during the mode transition. 

2.断市电/Disconnect the Grid Power:

断开市电电源，确保系统完全与市电隔离。

Disconnect the grid power supply to ensure the system is fully isolated from the grid.

3.改离网/Switch to Off-Grid Mode:

将系统模式从并网模式切换为离网模式。

Switch the system mode from grid-connected to off-grid.

4.开主机（若有并机）/Start the Main Unit:

重新启动主机，确保主机在离网模式下正常工作。

最后，启动从机，使整个系统在离网模式下正常运行。

Finally, start the slave unit to ensure the entire system operates normally in off-grid mode.

## 四、离网转并网操作流程 / Off-Grid to Grid-Connected Operation Procedure:

四、离网转并网操作流程 / Off-Grid to Grid-Connected Operation Procedure:

确保关机/Ensure Shutdown:

确保所有设备都已关闭，以防在转换过程中发生任何不稳定的情况。

Ensure all devices are shut down to prevent any instability during the transition.

改并网/Switch to Grid-Connected Mode:

将系统模式从离网模式切换回并网模式。

Switch the system mode from off-grid back to grid-connected.

3.上市电/Reconnect the Grid Power:

重新连接市电电源，确保系统可以正常接收市电输入。

Reconnect the grid power supply to ensure the system can properly receive grid input.

4.开主机/Start the Main Unit:

启动主机，使其在并网模式下正常运行。

Start the main unit, ensuring it operates correctly in grid-connected mode.

5.开从机/Start the Slave Unit:

启动从机，确保整个系统在并网模式下正常运行。

Start the slave unit to ensure the entire system operates normally in grid-connected mode.
