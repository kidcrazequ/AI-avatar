# 方案设计

> **级别**：🟢 基础
> **版本**：v1.0

---

## 技能说明

根据客户需求和场地条件，设计完整的工商业储能解决方案。

## 触发条件

当用户提到"出方案"、"方案设计"、"做个方案"、"储能方案"时触发。

## 输入

| 参数 | 是否必须 | 说明 | 示例 |
|------|---------|------|------|
| 客户行业 | 必须 | 客户所属行业 | 制造业、商业综合体 |
| 所在地区 | 必须 | 项目所在省/市 | 广东佛山 |
| 变压器容量 | 必须 | 用户侧变压器容量(kVA) | 1000kVA |
| 月用电量 | 建议提供 | 月均用电量(kWh) | 50000kWh |
| 最大需量 | 建议提供 | 月最大需量(kW) | 600kW |
| 电费账单 | 建议提供 | 近 3 月电费账单 | PDF/图片 |
| 场地条件 | 可选 | 可安装面积、室内/室外 | 室外 20㎡ |

如果用户未提供必须参数，按以下顺序询问：地区 → 行业 → 变压器容量。

## 执行流程

### Step 1：需求确认
- 确认所有必须参数已收集
- 查询 `knowledge/policy.md` 获取该地区电价政策
- 分析行业用电特征

### Step 2：容量配置
- 根据 `knowledge/solution-design.md` 中的方法论进行配置
- 确定储能功率(kW)和容量(kWh)
- 确定充放电策略（一充一放 / 两充两放）
- 验证变压器容量约束

### Step 3：设备选型
- 参考 `knowledge/technical.md` 和 `knowledge/competitors.md`
- 推荐 2-3 个设备方案（高/中/经济）
- 列出核心参数对比

### Step 4：收益测算
- 按 `skills/roi-calculation.md` 的方法进行测算
- 输出乐观/中性/保守三档
- 标注关键假设条件

### Step 5：风险评估
- 识别该项目的主要风险点
- 给出应对建议

### Step 6：输出方案

```markdown
- 用电现状分析

## 二、方案设计

### 2.2 设备选型
| 设备 | 推荐方案 | 备选方案 |
| 参数对比表 |

### 3.1 收益构成
| 收益来源 | 年收益(万元) |
### 3.2 财务指标
| 指标 | 乐观 | 中性 | 保守 |
| 静态回收期 | | | |
| IRR | | | |

## 四、风险评估
| 风险 | 影响 | 应对 |

## 五、实施建议
- 推荐实施步骤
- 关键时间节点
```

## 质量标准

- [ ] 容量配置有计算过程支撑
- [ ] 收益测算包含三档场景
- [ ] 设备选型有至少 2 个方案对比
- [ ] 风险评估不少于 3 个风险点
- [ ] 关键数据标注了来源
- [ ] 政策数据标注了时效性
