| 类别 | 要求 |
| --- | --- |
| 外观 | 1. 检查产品颜色外观为RAL7035（δE<1），喷塑工艺为桔纹（KPC），色板为阿克苏色板； |
| | 2. 整体机柜无明缝，外表防腐油漆层完整，无变形、裂纹、起泡等现象，全柜应少于5处补漆点，如果存在补漆点，要求90度正视无明显色差，允许45度侧视轻微的色差； |
| | 3. 机柜表面无损伤、变形、划痕等严重外观不良； |
| | 4. 机柜后门板（非铰链式）固定安装后，每组门板应整齐平整，不得有明显的歪斜现象； |
| | 5. 井组件安装正确，紧固螺钉需有效紧固与井组件无间隙； |
| | 6. 所有门板上及其它接地地点就应有清晰的标识，位置正确； |
| | 7. 变频器空机柜焊接要求：保证焊高大于2mm（不包含变频器底座）； |
| | 8. 门板安装附件安装螺栓需要点螺纹胶并严格按照装配力矩表 |
| | 9. 柜内覆铝锌板钣金件断面做喷锌防锈处理； |
| | 10. 序列号符合物流标签规范，内部有产品标签，序列号一致 |
