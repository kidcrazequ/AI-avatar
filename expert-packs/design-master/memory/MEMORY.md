## 使用规则

- **按主题组织**，不按时间堆砌（避免越来越长难检索）
- **每条记录尽量短**：一句话能说清就一句话，需要展开的写到 `wiki/` 里再链接过来
- **写入时机**：用户表达偏好 / 纠正你 / 拍板决策 / 给新案例时，立即写入
- **读取时机**：每次回答前，至少看一眼"纠偏记录"和"偏好"两节
